# Описание функционала

## endpoint /queue

### POST запрос

Формат json:

```json
{
  "name": "queue1",
  "message": "...."
}
```

Где:

* name - Имя очереди
* message - сообщение, которое нужно положить в эту очередь

### GET запрос

/queue?id=**queue_name** , где queue_name - имя очереди, из которой будет прочитано сообщение.
